API ENDPOINTS (READY)
================================================================================

GET /api/health
  → Server health check

POST /api/auth/login
  → User login

GET /api/secrets
  → List all secrets

POST /api/secrets
  → Create new secret
  Body: { name, service, value, description?, project_tags? }

GET /api/secrets/:id
  → Get secret with decryption

DELETE /api/secrets/:id
  → Delete secret

POST /api/secrets/:id/rotate
  → Rotate secret (archive old value)

GET /api/tokens
  → List API tokens

POST /api/tokens
  → Create token
  Body: { name, allowed_tags? }

DELETE /api/tokens/:id
  → Revoke token
